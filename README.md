# HOW TO COMPILE

1. Install python (newer releases)
     Link: https://www.python.org/downloads/

2. Inside Visual Studio Code (or whatever IDE) run "pip install pyinstaller"
   
3. Now run "pyinstaller --onefile --noconsole --icon=macro.ico main.py"
